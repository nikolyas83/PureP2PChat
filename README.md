# PureP2PChat

## Опис

PureP2PChat - це простий peer-to-peer чат додаток, що складається з Android додатку та Python сервера для тестування. Додаток дозволяє спілкуватись у локальній мережі або через інтернет використовуючи релей сервер.

## Архітектура

### Режими роботи

1. **Локальна мережа** (Local Network):
   - Використовує UDP broadcast для виявлення користувачів у локальній мережі
   - TCP з'єднання для передачі повідомлень
   - Порт 8888 для TCP/UDP

2. **Серверний режим** (Server Mode):
   - Підключається до релей сервера через TCP
   - Порт 9999 для релей з'єднання
   - Дозволяє спілкуватись через інтернет

### Компоненти

- **Android додаток**: Клієнтська частина з графічним інтерфейсом
- **Python сервер**: Тестовий інструмент з GUI для налаштувань та чату

## Вимоги

### Для Android додатку
- Android 5.0+ (API 21+)
- Інтернет дозвіл для віддалених з'єднань

### Для Python сервера
- Python 3.x
- Tkinter (зазвичай включений у Python)
- Socket library (включена у стандартну бібліотеку)

## Встановлення та запуск

### Android додаток

1. Відкрийте проект в Android Studio
2. Підключіть Android пристрій або створіть емулятор
3. Зберіть та встановіть APK

```bash
./gradlew assembleDebug
```

### Python сервер

```bash
python test_chat.py
```

## Налаштування

### Локальна мережа
- Оберіть режим "Локальна мережа"
- Введіть ім'я користувача
- Додаток автоматично знайде інших користувачів у мережі

### Віддалене підключення
- На ПК запустіть Python сервер з увімкненим "Запустити як сервер"
- На телефоні оберіть режим "Сервер"
- Введіть адресу сервера (наприклад: your-ddns-domain.com)
- Порт 9999 має бути відкритий та перенаправлений у роутері

## Проблеми та рішення

### Не працює віддалене підключення
1. Перевірте firewall на ПК - дозвольте порт 9999
2. Перевірте перенаправлення портів у роутері
3. Перевірте DDNS налаштування
4. Мобільний провайдер може блокувати вхідні з'єднання

### Помилки у Python
- Якщо виникає помилка прив'язки порту, переконайтеся що попередній процес зупинено
- Налаштування зберігаються у `settings.json`

## Безпека

- Додаток не використовує шифрування повідомлень
- Не передавайте конфіденційну інформацію
- Використовуйте тільки у довірених мережах
